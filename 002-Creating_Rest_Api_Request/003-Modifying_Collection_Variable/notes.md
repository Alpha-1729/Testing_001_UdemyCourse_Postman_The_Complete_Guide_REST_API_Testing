# Modifying Collection Variable

- Refer the `assets/` folder for screenshots and GIFs.
- Difference between **Initial Value** and **Current Value** in collection variable.
  - When you share a collection, only the **Initial Value** is shared.
  - **Current Value**
    - Used in the actual request.
    - Private — nobody else can see it.
  - **Initial Value**
    - Use this as a placeholder for passwords.
- If the variable color in the request is **red** — Postman cannot resolve the variable.
- Hover over a variable to see its initial value, current value, and scope.
- Hide left sidebar — click the icon on the bottom left corner.
- To delete a collection variable — refer the `assets/` folder for GIF.
