# Working With HTTP Headers

- Click on the `Headers` tab below the request to view request headers.
- Responses may also contain headers.
- You can add custom headers in the request.
  - Refer the `assets/` folder for screenshots.
- To edit a predefined header created by Postman, you have to redefine it.
  - To remove the predefined `Authorization` header.
    - Go to `Authorization` tab → set to `No Auth` or `Inherit from Parent`.
    - Then redefine the `Authorization` header manually.
