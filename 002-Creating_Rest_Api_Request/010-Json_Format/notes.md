# JSON Format

- **JSON** (JavaScript Object Notation) — a lightweight data format for exchanging data.
- Human-readable and easy to understand.
- Data is stored in **key-value pairs**.
  - `"name": "John"`
- Supports data types — strings, numbers, booleans, arrays, objects, and null.
- Lightweight compared to XML — smaller file size, faster data transfer.
- Language independent — works with almost all programming languages.
- Syntax rules.
  - Curly braces `{}` hold objects.
  - Square brackets `[]` hold arrays.
  - Comma `,` separates data.
