# POST Requests

- `POST` request is used to send data to the server to create a new resource.
- Creating a cart.
  - `POST` <https://simple-grocery-store-api.glitch.me/carts>
- Get a cart.
  - `GET` <https://simple-grocery-store-api.glitch.me/carts/:cartId>
- Get cart items.
  - `GET` <https://simple-grocery-store-api.glitch.me/carts/:cartId/items>
- Successful response status codes.
  - `201 Created` — request succeeded and a new resource was created.
  - <https://developer.mozilla.org/en-US/docs/Web/HTTP/Status#successful_responses>
- HTTP response status codes reference.
  - <https://developer.mozilla.org/en-US/docs/Web/HTTP/Reference/Status#successful_responses>
