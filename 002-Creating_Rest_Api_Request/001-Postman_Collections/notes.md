# Postman Collections

- **Collections** — a folder that stores multiple requests.
- Adding a request to a collection.
  - Click on the `Save` button near the request.
  - Add a request name.
  - Create a collection.
  - Click on `Save`.
- If you edit a request, click the `Save` button to save changes.
- **Save As** option — edit the current request and save it as a new request.
  - Refer the `assets/` folder for screenshots.
