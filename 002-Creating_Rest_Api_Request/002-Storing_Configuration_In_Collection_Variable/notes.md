# Storing Configuration In Collection Variable

- **Collection Variable** — stores a value that can be reused across all requests in a collection.
  - Example: If `baseUrl` changes, update it in one place instead of all 10 requests.
  - Refer the `assets/` folder for screenshots.
- Syntax for a variable in Postman.
  - `{{baseUrl}}`
- A variable created under a collection is available for all requests under that collection.
