# GET Vs POST

- **GET** — used to fetch/retrieve data from the server.
  - Data is sent via URL as query parameters.
  - No request body.
  - Can be bookmarked and cached.
  - Example: `GET {{baseUrl}}/products`

- **POST** — used to send data to the server to create a new resource.
  - Data is sent in the request body (not visible in URL).
  - Cannot be bookmarked or cached.
  - Example: `POST {{baseUrl}}/carts`
