# First Postman Request

- Simple Grocery Store API Documentation.
  - <https://github.com/vdespa/Postman-Complete-Guide-API-Testing/blob/main/simple-grocery-store-api.md>
  - Document is also attached in the `assets/` folder.
- Get the status of the grocery store API.
  - On Postman
    - `GET` <http://simple-grocery-store-api.online/status>
    - `GET` <https://simple-grocery-store-api.glitch.me/status>
  - On Browser
    - <http://simple-grocery-store-api.online/status>
    - <https://simple-grocery-store-api.glitch.me/status>
- Click on the `Console` button on the bottom left corner to get more info like response time.
  - Refer the `assets/` folder for screenshots.
- If company network is blocking the request, try the Postman web version.
- **Endpoint** — different addresses within the API where we can find information.
- Encoding for white space character → `%20`
