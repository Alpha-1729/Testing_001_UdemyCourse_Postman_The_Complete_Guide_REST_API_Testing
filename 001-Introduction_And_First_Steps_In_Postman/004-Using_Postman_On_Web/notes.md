# Using Postman On Web

- **Cloud Agent**
  - Browser sends request to Postman cloud agent.
  - Cloud agent sends request to the API.
  - Has a limitation on the number of requests that can be sent.
  - Refer the `assets/` folder for screenshots.
- **Desktop Agent**
  - A small program installed on the computer, runs in the background.
  - Initiates requests from the locally running Postman.
  - Visible in the system tray (bottom right corner).
  - <https://www.postman.com/downloads/postman-agent/>
- **Browser Agent**
  - Sends requests through the browser.
  - Works only sometimes due to browser limitations.
- Agent option is only shown in the Postman web version.
