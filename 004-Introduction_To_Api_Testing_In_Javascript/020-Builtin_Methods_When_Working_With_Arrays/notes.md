# Builtin Methods When Working With Arrays

- `Array.length` — returns the number of elements in an array.
  - <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/length>
- `Array.pop()` — removes and returns the last element of an array.
  - <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/pop>
- `Array.push()` — adds one or more elements to the end of an array.
  - <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/push>
- `Array.slice()` — returns a shallow copy of a portion of an array without modifying the original.
  - <https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Array/slice>
