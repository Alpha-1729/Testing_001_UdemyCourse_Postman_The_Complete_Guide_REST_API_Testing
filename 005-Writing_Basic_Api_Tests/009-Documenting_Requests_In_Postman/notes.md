# Documenting Requests In Postman

- Postman allows adding documentation at the **request level**.
  - Documentation can also be added at the **folder** and **collection** level.
  - Refer the `assets/` folder for GIF.
- Add API documentation in Postman.
  - <https://learning.postman.com/docs/publishing-your-api/documenting-your-api/#documenting-a-collection>
- Supports **Markdown** formatting in documentation.
- Documentation is visible when sharing or publishing a collection.
